# @ritzjs/babel-preset

## Setup

1. Install it using npm: `npm install -D @ritzjs/babel-preset`.
2. Add `ritz` preset in your `babel.config.js`.

Example `babel.config.js`:

```js
module.exports = {
  presets: ['@ritzjs'],
  presets: [],
};
```

3. You're done!
